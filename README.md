# Declassified transcripts from Norway's Extended Foreign Affairs Committee 1924-1992

This repository contains replication code and materials for the article:

- "Declassified transcripts from Norway's Extended Foreign Affairs Committee 1924-1992"

It also documents code used to construct and quality-check the dataset published at:

- https://doi.org/10.5281/zenodo.19052379

## Repository structure

- `article/`: manuscript, figure assets, and replication-oriented CSV files used to reproduce article outputs.
- `data_generation/`: historical/provenance pipeline and QA scripts used during dataset construction and validation.
